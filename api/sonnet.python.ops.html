sonnet.python.ops.nest module¶

Utility functions for dealing with nested structures of Tensors.

These complement nest.flatten and nest.pack_sequence_as from the core TF distribution.

sonnet.python.ops.nest.flatten_iterable(structure)[source]¶

Returns a flat list from a given nested structure.

If structure is not a sequence, tuple, or dict, then returns a single-element list: [structure].

In the case of dict instances, the sequence consists of the values, sorted by key to ensure deterministic behavior. This is true also for OrderedDict instances: their sequence order is ignored, the sorting order of keys is used instead. The same convention is followed in pack_sequence_as. This correctly repacks dicts and OrderedDicts after they have been flattened, and also allows flattening an OrderedDict and then repacking it back using a correponding plain dict, or vice-versa. Dictionaries with non-sortable keys cannot be flattened.

Args: structure: an arbitrarily nested structure or a scalar object. Note, numpy arrays are considered scalars.

Returns: A Python list, the flattened version of the input.

Raises: TypeError: The structure is or contains a dict with non-sortable keys.

sonnet.python.ops.nest.pack_iterable_as(structure, flat_iterable)[source]¶

Returns a given flattened sequence packed into a given structure.

If structure is a scalar, flat_iterable must be a single-element list; in this case the return value is flat_iterable[0].

If structure is or contains a dict instance, the keys will be sorted to pack the flat sequence in deterministic order. This is true also for OrderedDict instances: their sequence order is ignored, the sorting order of keys is used instead. The same convention is followed in pack_sequence_as. This correctly repacks dicts and OrderedDicts after they have been flattened, and also allows flattening an OrderedDict and then repacking it back using a correponding plain dict, or vice-versa. Dictionaries with non-sortable keys cannot be flattened.

Args: structure: Nested structure, whose structure is given by nested lists, tuples, and dicts. Note: numpy arrays and strings are considered scalars. flat_iterable: flat sequence to pack.

Returns: packed: flat_iterable converted to have the same recursive structure as structure.

Raises: ValueError: If flat_iterable and structure have different element counts. TypeError: structure is or contains a dict with non-sortable keys.
